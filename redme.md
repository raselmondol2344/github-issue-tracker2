 Qustion : 01  = >  What is the difference between var, let, and const?
 Answer : 
 ** var => পুরোনো নিয়ম, variable বানানো যায়। একই নাম দিয়ে আবার বানানোও যায়। value চাইলে  পরে পরিবর্তন করা যায়।

 ** let => নতুন নিয়ম, variable বানানো যায়। value চাইলে পরে পরিবর্তন করা যায়, কিন্তু একই নাম দিয়ে আবার declare করা যাবে না।

 ** const => একবার value ঠিক করলে পরে আর পরিবর্তন করা যায় না।



Qustion : 02 => What is the spread operator (...)?
Answer : 
Spread operator ... হলো JavaScript এ array বা object-এর সব value বা property ফেলে নতুন array/object বানানোর জন্য ব্যবহার করা হয়।




Qustion : 03 => What is the difference between map(), filter(), and forEach()?
Answer : 

** forEach => শুধু loop চালায়
** map => নতুন array বানায় (প্রত্যেক element modify করে)
** filter => নতুন array বানায় (শর্ত মিলে এমন element নিয়ে)



Qustion : 04 => What is an arrow function?
Answer : 
Arrow function হলো ছোট এবং সরলভাবে function লিখার উপায়, যা ()=>{} দিয়ে বানানো হয়।



Qustion : 05 => What are template literals?
Answer : 
Template literals হলো JavaScript এ string বানানোর নতুন উপায়, যা ` ` (backtick) এর মধ্যে লেখা হয় এবং ${} ব্যবহার করে variable বা expression সহজে বসানো যায়।
